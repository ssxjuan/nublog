# ¿Qué es TradingView?

TradingView es una plataforma web para el análisis técnico de mercados financieros, con herramientas para el seguimiento de cotizaciones y gráficos. Esta plataforma se ha convertido en una de las mejores opciones para los comerciantes de todos los niveles de habilidad, ya que ofrece una amplia gama de herramientas para el análisis técnico, una interfaz intuitiva y una comunidad de usuarios activos. 

<ImageAdd source="https://live.staticflickr.com/65535/51709052698_5d44fcbc11_o.png"/>

## Funcionalidades de TradingView

TradingView ofrece una variedad de herramientas y recursos para ayudar a los operadores a tomar mejores decisiones. Estas herramientas incluyen gráficos en tiempo real, herramientas de análisis técnico, noticias de mercado, indicadores, alertas de precios y más. Además, TradingView ofrece una amplia gama de productos, desde herramientas gratuitas hasta productos de suscripción de pago.

<ImageAdd source="https://s3.tradingview.com/snapshots/n/n5dNAFv1.png"/>

## ¿Cómo funciona TradingView?

TradingView se basa en una plataforma web intuitiva que permite al usuario personalizar gráficos, indicadores y alertas para ayudar a monitorizar los mercados de forma eficaz. Los usuarios también pueden tener una visión más profunda de los mercados financieros con la ayuda de herramientas de análisis técnico como el análisis de patrones de gráficos, el análisis de tendencias y la identificación de señales de compra/venta. 

Además, los usuarios pueden interactuar con la comunidad de TradingView para obtener ideas comerciales, recibir consejos y feedback de otros usuarios y compartir sus propias ideas y análisis.
